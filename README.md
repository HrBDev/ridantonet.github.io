## 📥 درباره پروژه

یه ابزار ساده و کاربردی که لینک یوتیوب رو میگیری، ویدیو رو با بهترین کیفیت دانلود میکنی. همه چیز با **GitHub Actions** اجرا میشه — بدون نیاز به سرور یا نصب نرم‌افزار خاص.

> ⚡ فایل‌ها **۵ دقیقه** پس از دانلود خودکار حذف میشن. سریع دانلود کن.

---

## ✨ ویژگی‌ها

- **🎯 لینک بده، دانلود کن** — فقط URL رو وارد کن، بقیه خودکاره
- **🎬 کیفیت دلخواه** — از 480p تا 4K، یا فقط صدا (MP3)
- **📝 زیرنویس** — فارسی و انگلیسی (اگه موجود باشه)
- **🔒 محافظت با رمز** — می‌تونی روی فایل خروجی رمز بذاری
- **📦 مدیریت فایل‌های حجیم** — ویدیوهای بلند خودکار تکه‌تکه میشن
- **🌐 بدون تحریم** — استفاده از Cloudflare WARP برای دانلود
- **🧹 پاکسازی خودکار** — فایل‌ها بعد از ۵ دقیقه حذف میشن
- **📱 کاملاً ریسپانسیو** — با موبایل، تبلت و دسکتاپ کار میکنه

### از طریق گیت‌هاب هم می‌تونی استفاده کنی:

1. برو به تب **Actions** توی این ریپازیتوری
2. روی **Khashayar YouTube Downloader** کلیک کن
3. **Run workflow** رو بزن، لینک و تنظیمات رو وارد کن
4. بعد از اتمام، فایل‌ها توی پوشه `videos/` قرار می‌گیرن
